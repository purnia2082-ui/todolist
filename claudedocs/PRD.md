# PRD: 할일 관리 앱

## 1. 개요

### 프로젝트 목적
개인 사용자가 일상적인 할일을 체계적으로 기록하고 완료 여부를 추적하여 생산성을 높일 수 있도록 돕는 할일 관리 앱

### 타겟 사용자
- 주요 사용자: 일상적인 업무와 개인 과업을 관리하고 싶은 20~40대 개인 사용자
- 특성: 별도의 학습 없이 즉시 사용 가능한 단순한 도구를 선호하며, 브라우저에서 바로 접근 가능한 환경을 원함

### 핵심 가치 제안
복잡한 설정 없이 할일을 즉시 기록하고, 완료 상태를 한눈에 파악하여 오늘 해야 할 일을 놓치지 않는다.

---

## 2. 기능 요구사항

### 기능 1: 할일 추가 및 삭제

**사용자 스토리**
> 개인 사용자로서, 할일을 텍스트로 입력하여 목록에 추가하고 필요 없는 항목은 삭제하고 싶다, 내가 해야 할 일을 빠짐없이 기록하고 불필요한 항목을 제거하여 목록을 깔끔하게 유지하기 위해

**수용 기준**
- [ ] 사용자가 텍스트 입력 후 등록 버튼을 누르거나 Enter 키를 입력하면 해당 텍스트가 할일 목록에 즉시 추가된다
- [ ] 내용이 없는 빈 텍스트(공백만 포함된 경우 포함)는 추가되지 않으며, 사용자에게 입력 필요 안내 문구가 표시된다
- [ ] 각 할일 항목에는 삭제 버튼이 존재하며, 버튼 클릭 시 해당 항목만 목록에서 제거된다
- [ ] 삭제 후 나머지 항목의 순서와 상태는 변경되지 않는다
- [ ] 할일 텍스트는 최대 200자까지 입력 가능하며, 초과 시 더 이상 입력되지 않는다

---

### 기능 2: 할일 완료 체크

**사용자 스토리**
> 개인 사용자로서, 완료한 할일에 체크 표시를 하고 싶다, 무엇을 끝냈는지 한눈에 확인하고 성취감을 느끼기 위해

**수용 기준**
- [ ] 각 할일 항목에는 체크박스가 존재하며, 클릭 시 해당 항목이 완료 상태로 전환되고 시각적으로 구분된다 (예: 텍스트에 취소선 표시)
- [ ] 완료 상태의 항목을 다시 클릭하면 미완료 상태로 되돌아간다 (토글 방식)
- [ ] 완료 상태는 페이지를 새로고침하더라도 유지된다
- [ ] 완료 상태 전환은 즉시 반영되며, 별도의 저장 버튼을 누를 필요가 없다

---

### 기능 3: 할일 목록 조회

**사용자 스토리**
> 개인 사용자로서, 전체 할일 목록과 완료/미완료 항목을 구분하여 조회하고 싶다, 현재 남은 과업과 완료된 과업을 명확히 파악하기 위해

**수용 기준**
- [ ] 앱 진입 시 저장된 모든 할일 항목이 목록 형태로 표시된다
- [ ] 목록 상단 또는 탭 영역에 전체 / 미완료 / 완료 필터가 제공되며, 선택한 필터에 해당하는 항목만 표시된다
- [ ] 전체 할일 수, 완료된 할일 수가 숫자로 표시되어 진행 현황을 즉시 파악할 수 있다
- [ ] 할일 항목이 하나도 없을 경우 "등록된 할일이 없습니다" 안내 문구가 표시된다

---

## 3. 비기능 요구사항

### 성능
- 할일 추가, 삭제, 완료 체크 등 모든 인터랙션은 사용자 행동 후 1초 이내에 화면에 반영된다
- 할일 항목이 100개 이상이더라도 목록 조회 및 필터 전환이 지연 없이 동작한다

### 보안
- 사용자의 할일 데이터는 해당 사용자의 브라우저 로컬 환경에만 저장되며, 외부로 전송되지 않는다

### 접근성
- 마우스 없이 키보드만으로 할일 추가, 삭제, 완료 체크, 필터 전환이 모두 가능하다
- 모바일(320px 이상) 및 데스크톱(1280px 이하) 화면 모두에서 레이아웃이 깨지지 않는다
- 색상만으로 완료/미완료를 구분하지 않고, 텍스트 또는 아이콘을 병행하여 색약 사용자도 상태를 인식할 수 있다

---

## 4. 제약사항

### 기술적 제약
- 별도 회원가입 및 로그인 기능 없이 동작해야 한다 (MVP 범위)
- 인터넷 연결 없이도 기본 기능이 동작해야 한다

### 비즈니스 제약
- MVP는 핵심 기능 3개만 포함하며, 추가 기능은 사용자 피드백 수집 이후 결정한다

---

## 5. 성공 지표
- 앱 최초 진입 후 60초 이내에 첫 번째 할일을 추가할 수 있다
- 필터 기능을 사용하여 미완료 항목만 조회하기까지 사용자 클릭이 2회 이내로 가능하다
- 페이지 새로고침 후에도 이전에 입력한 할일 목록과 완료 상태가 동일하게 유지된다

---

## 향후 확장 (MVP 이후)
- 할일 마감일 설정 및 날짜별 필터 조회
- 카테고리 또는 태그 분류 기능
- 할일 우선순위 지정 (높음 / 보통 / 낮음)
- 완료된 할일 일괄 삭제 기능
- 할일 순서 직접 변경 (드래그 앤 드롭)
